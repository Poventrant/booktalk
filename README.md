# booktalk
## 简介
个人毕设项目，一个关于书籍借阅交流的社区网站  
前端 bootsrap + jquery   
后端 express mongoDB  
### 运行
```
git clone https://github.com/fuckingcode/booktalk.git
npm install
启动mongoDB
在config.js里将相关配置修改成自己的本地配置
运行 npm start  (node .bin/www)
或者 npm run super (supervisor .bin/www) 可使在修改后台文件时不必重启项目
```
